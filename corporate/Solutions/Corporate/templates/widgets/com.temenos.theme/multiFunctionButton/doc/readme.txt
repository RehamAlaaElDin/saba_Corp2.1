This widget lets you control the classes on other elements in the page when you click the button using the widget.

	1) You can remove classes from various element
	2) You can toggle classes on elements
	3) You can add classes to elements
	
In addition, you can also invoke a sound file (using the id of the sound element) and invoke some javascript.  The 
parameter passed to the javascript function is an object of the parameters used above.

It is also possible to choose whether you want the button to run its rules (ie. go back to the server) or not.
